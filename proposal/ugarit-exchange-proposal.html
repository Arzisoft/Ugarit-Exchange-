Ugarit Exchange

The Vision

Ugarit Exchange is a licensed, regulated digital financial platform enabling individuals and businesses across the Middle East and Africa to exchange currencies, send international remittances, manage digital wallets, trade crypto, and access virtual debit cards — all from a single, premium interface.

Architecture Overview

Frontend Architecture

  • 95 source files across components, pages, stores, and utilities
  • 23 routes (21 static + 2 dynamic)
  • Server & Client components optimized for performance
  • Route groups for shared layouts without URL impact
  • Mock data layer with localStorage persistence

Production-Ready For

  • Client demonstration — All flows work end-to-end
  • Investor presentations — Professional UI/UX
  • White-label deployment — Single config swap
  • Backend integration — Replace mock stores with API calls
  • Vercel deployment — One-click deploy ready

White-Label Capability

The entire platform identity is driven by a single configuration file. Switching from Ugarit Exchange to Palremit (or any future brand) requires changing one environment variable: NEXT_PUBLIC_BRAND=palremit. This swaps the brand name, logo, colors, tagline, supported currencies, and footer credits automatically.
